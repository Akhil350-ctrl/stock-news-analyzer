# Stock News Analyzer 📰📈

This project analyzes stock market news headlines and provides sentiment insights.

## Features
- News Sentiment (Positive / Negative)
- Market Mood Indicator
- Basic Buy/Sell Bias

## Status
🚀 Actively developing
Update: Improving sentiment analysis logic
## How It Works
This tool uses simple keyword-based sentiment analysis to classify stock news as positive, negative, or neutral.
